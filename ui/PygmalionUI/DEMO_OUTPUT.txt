NAVIGATION:
-----------
• Use ARROW KEYS to navigate between menu items
• Press ENTER to select an item
• Press ESC to return to the main menu from any view
• Click with MOUSE to select items (if supported by terminal)

COLOR SCHEME:
-------------
• Green background matching catapeltUI style
• White text on green for normal items
• Black text on bright green for focused items
• Professional terminal appearance

FEATURES:
---------
✓ Full keyboard navigation
✓ Mouse support
✓ Dialog boxes for data entry
✓ List views for displaying data
✓ Add/Remove functionality
✓ ESC key navigation
✓ Clean, professional UI
